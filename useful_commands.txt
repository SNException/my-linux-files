Change the root directory (e.g to create jail):
    $ chroot <dirname>

To access root only dir:
    $ sudo su
    $ cd <dirname>

Make directory root only:
    $ chown root:root <dirname>
    $ chmod 700 <dirname>

Cat file without excecuting vt codes
    $ cat -v somefile

Gets logged in user:
    $ logname

Strip filename from path (including extension)
    $ basename /some/path/to/a/file.txt .txt

Strip dirname from path:
    $ dirname /some/path/to/a/file.txt

Show last logins:
    $ last
    $ last | head
    $ sudo cat /var/log/auth.log
    $ lastlog

List all opened sockets and their processes:
    $ lsof -i

List all installed packages:
    $  dpkg -l

Print the printable characters of a file in a block of four (useful for binaries):
    $ strings <filename>

Simple way to append to a file:
    $ echo "Hi" >> <filename>

Simple way to overwrite a file content:
    $ echo "Hi" > <filename>

Get only the names of all running processes:
    $ ps axch -o cmd

List all installed fonts
    $ fc-list

Hexdump the first 100000 bytes of a device:
    $ sudo hexdump -C -v -n 100000 /dev/<device_name>

Make file system (e.g FAT32):
    $ sudo mkfs.vfat -F 32 /dev/<device_partition>

Check partitions:
    $ sudo fdisk -l
    $ lsblk -f

Partition disk:
    $ sudo fdisk /dev/<device_name_not_partition>

Wipe entire device with zeros:
    $ sudo dd if=/dev/zero of=/dev/<device_name_not_partition> bs=1M status="progress"

Dump file content to stdout in reverse (oposite of cat):
    $ tac <filename>

Math with awk (floats cutting of after the dot (integer round)):
    $ awk -v var1=2323.4343 -v var2=43.12000 'BEGIN {print var1/var2}' | cut -d '.' -f1

Math with awk (floats):
    $ awk -v var1=10.5 -v var2=4.3 'BEGIN {print var1/var2}'

Print total user count:
    $ echo -n "Total user count: " && awk -F ":" '{print $1}' /etc/passwd | wc -l

Filtering via an if-statement with awk:
    $ lsblk | awk '{if ($NF != "disk" && $NF != "part" && $NF != "MOUNTPOINT") print $NF}'

Print every line wich does NOT start with a specific string via awk:
    $ awk '!/^alias/ {print}' .bash_aliases

Print last column with awk:
    $ df | awk '{print $NF}'

Print specific line with awk:
    $ awk '{if (NR == 10) print}' <filename>

Emulate cat with awk:
    $ awk '{print}' <filename>

Print every line of a file which starts with '#':
    $ awk '/^#/ {print}' <filename>

For loop with awk:
    $ awk 'BEGIN {for(i = 0; i < 100; ++i) print i*i}'

Print all users with awk (changing column delimitter to a colon):
    $ cat /etc/passwd | awk -F ":" '{print $1}'

Basic awk syntax:
    $ awk 'BEGIN {print "About to process..."} {print $1} END {print "Done processing"}' <filename> 

awk hello world:
    $ awk 'BEGIN {print "Hello, world!"}'

Very useful way of using awk (make use of xargs):
    $ some_output_with_multiple_lines | xargs | awk '{print $1}'

Show type of all files which also have quotes in their names:
    $ find . -type f -print0 | xargs -0 fileS

Play mp4/3 file with ffplay
    $ ffplay -fs -autoexit <filename>

For loop range:
    $ for i in $(seq 10); do echo $i; done

Convert mp4 (video) file to mp3 (audio) file via ffmpeg:
    $ ffmpeg -i file.mp4 file-mp3

Add directory to path (put it inside .bashrc):
    $ export PATH="$PATH:/path/to/dir"

Replace a string with something else:
    $ echo $SOMETHING | sed 's/What/With/g'

Store command result in variable:
    $ OUTPUT=$(sensors | awk '/^Core/ {print $3}')

Create environment variable (put it in '~/.bashrc' or put in '/etc/environment' to make it permanent)
    $ export SOME_NAME="some value"

Print environment variable (e.g random number)
    $ echo $RANDOM

Download something from the internet (e.g an image)
    $ wget www.for.com/bar.jpg

Execute multiple commands as su:
    $ sudo -s eval 'ls -aril && echo "foobar" && cat foo.txt | head -10'

Show services of systemd:
    $ systemctl list-units

List autostart programs (root):
    $ ls /etc/xdg/autostart

List autostart programs (user-specific):
    $ ls ~/.config/autostart

Clear an alias:
    $ unalias <name>

Set an alias (e.g rot13):
    $ alias rot13="tr 'A-Za-z' 'N-ZA-Mn-za-m'"

Grep something which is not equal to the given string (e.g list only block devices):
    $ find /dev/ | xargs file | grep -v character | grep -v directory | grep -v symbolic

    $ lsusb

List open files information:
    $ lsof

    $ lscpu

List hardware:
    $ lshw

List block devices (drives):
    $ lsblk

    $ dmesg

    $ uname -r

Play angband inside terminal:
    $ angband -mgcu

Watch a command result:
    $ watch sensors

    $ systemd-analyze blame

    $ find . | grep -i <filename> | xargs vim

    $ sudo chkrootkit
    $ sudo rkhunter --sk --check

Get sha256 from every file in directory:
    $ find <dirname> | xargs sha256sum

Create sequence of files
    $ seq 10 | xargs touch

Count all lines recursively:
    $ find *.* | xargs wc -l

Shutdown now:
    $ shutdown now

Kill (Terminate) process by name:
    $ killall <processname>

Good was to list a directory:
    $ ls -aril

    $ uptime

    $ grep -irn "static" .
